# 创意激发技巧库

创意激发者可参考以下技巧来生成创意。

## 跨界联想技巧

### SCAMPER 法
- **S**ubstitute（替代）— 能用什么替代现有方案？
- **C**ombine（组合）— 能把两个不相关的东西组合吗？
- **A**dapt（适配）— 其他领域的方案能适配过来吗？
- **M**odify（修改）— 放大/缩小/改变某个属性会怎样？
- **P**ut to other use（新用途）— 这个东西还能用在哪？
- **E**liminate（消除）— 去掉某个环节会怎样？
- **R**everse（反转）— 反过来做会怎样？

### 类比思维模板
- "这就像 [领域] 的 [模式]"
- "如果 [知名公司] 做这个，他们会怎么做"
- "如果这个产品是 [动物/食物/电影]，它是什么"

## 第一性原理技巧

### 5 Whys
连续问 5 次"为什么"，直到找到根本原因。

```
问题: 用户不付费
→ 为什么? 因为觉得不值
→ 为什么觉得不值? 因为免费内容已经够看
→ 为什么够看? 因为免费章节太多
→ 为什么太多? 因为我们的免费阈值设太高
→ 根因: 免费策略需要调整
```

### 减法思维
列出产品的所有组成部分，逐一去掉，思考没有它会怎样。

## 反直觉想法模板

```markdown
**想法: [一句话描述]**
- 常识认为: [大多数人怎么想]
- 反过来想: [如果反过来呢]
- 为什么可能有效: [逻辑支撑]
- 最大障碍: [什么会阻止它成功]
- 如果成功: [带来的价值]
- 验证方式: [怎么测试这个想法]
```
