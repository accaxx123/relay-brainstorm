# 市场分析框架

商业分析师在做市场分析时应使用以下框架。

## TAM/SAM/SOM 模型

- **TAM (Total Addressable Market)** — 整个市场的总规模
- **SAM (Serviceable Available Market)** — 你的产品能服务的市场部分
- **SOM (Serviceable Obtainable Market)** — 你实际能获取的市场份额

```markdown
### 市场规模
- TAM: [金额] — [定义]
- SAM: [金额] — [定义]
- SOM: [金额] — [定义]
```

## 成本结构

```markdown
### 固定成本（每月）
- 服务器: ¥X
- 人力: ¥X
- 工具/服务: ¥X
- 合计: ¥X

### 可变成本（每笔交易）
- 支付手续费: X%
- 内容成本: ¥X/单元
- 客服成本: ¥X/用户
```

## 收入模型

```markdown
### 收入来源
1. [来源 1] — 定价: ¥X — 预期用户: Y — 月收入: ¥Z
2. [来源 2] — 定价: ¥X — 预期用户: Y — 月收入: ¥Z

### 盈亏平衡点
- 月固定成本: ¥X
- 单用户贡献: ¥Y
- 盈亏平衡用户数: X/Y = Z 人
```
